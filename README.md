# Jira Low-Cost — .NET + Angular

**Prototype en cours de développement.**

Ce projet est une exploration de l'écosystème .NET.

## 🎯 Objectif

Reproduire la logique métier de mon projet NestJS (Jira Low-Cost) en .NET, afin de comparer les deux écosystèmes et de monter en compétence sur la stack Microsoft.
https://github.com/lucianorakotoarivony-netizen/Jira-low-cost

## 🧱 Stack

- **Backend** : .NET 10, Entity Framework Core, PostgreSQL
- **Frontend** : Angular 19+
- **Authentification** : JWT (Identity)
- **Conteneurisation** : Docker

## 🚦 Statut

Ce projet est au **tout début** de son développement. Les fonctionnalités suivantes sont opérationnelles :
- Authentification (login)
- Modèle de tâche (TaskItem)
- API REST (GET, POST)

Le reste est en cours d'implémentation.
